# RewardHub – Jekyll GitHub Pages Site

A fully featured rewards/loyalty platform built with Jekyll and deployed via GitHub Pages.

## Quick Start

### 1. Prerequisites
- Ruby 3.x + Bundler
- Git

### 2. Install dependencies
```bash
bundle install
```

### 3. Run locally
```bash
bundle exec jekyll serve --livereload
# Visit http://localhost:4000
```

### 4. Deploy to GitHub Pages
Push to `main` branch — the GitHub Actions workflow handles the rest.
